## A simple NodeJS Webapplication that allows you to Post Tweets

### Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/SwitzerChees/m183-minitwitter
   ```
2. Install NPM packages
   ```sh
   yarn install
   # or
   npm install
   ```
3. Run the app
   ```sh
   yarn start
   # or
   npm start
   ```
4. Open the app in your browser
   ```sh
   http://localhost:3000
   ```
